# QuantumErrorCorrectionCodes

A university research project. With my teammates, we investigated from scratch the topic of quantum computing, algorithms on a quantum computer, errors in quantum circuits and the way we could correct them.


## File structure

Here is a breakdown of what the files in this repo accomplishes : 
- `testing.py` : Used to run code from every other code to obtain graphs and results.
- `error_model.py` : Implementation of the ErrorModel class outlined in the report.
- `corrected_circuit.py` : Implementation of the QuantumCircuitCorrected class outlined in the report.
